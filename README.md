# 💵 Cotação Financeira

Projeto desenvolvido com **HTML, CSS e JS** com o objetivo de elaborar uma UI que consulta cotações financeiras em tempo real através de uma API externa.
O projeto inclui estilização com CSS, consumo de API com JS e persistência de histórico de consultas via `localStorage`.

> Curso Gestão da Tecnologia da Informação - Fatec Itu

## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3
- Javascript (Vanilla JS)
